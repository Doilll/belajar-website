Introduction

Java adalah bahasa pemrograman yang dibuat oleh james gosling dan sudah ada sejak tahun 1995. Ini bukan kopi ataupun pulau jawa, tapi alat keren yang digunakan buat bikin aplikasi dan game kayak Minecraft. Jadi, bayangin aja Java itu seperti pensil serbaguna buat bikin berbagai macam karya di dunia digital.

Java menggunakan konsep Object-Oriented Programming (OOP) yang bikin kode lebih terstruktur dan mudah dipelihara.

Kode yang ditulis di Java bisa jalan di berbagai platform (Windows, Mac, Linux, dll). Istilah kerennya adalah "Write Once, Run Anywhere" (WORA). Artinya, kamu nulis sekali, terus bisa dijalanin di mana saja.

Tipe Data dan Variabel

Bayangin kamu punya kotak yang bisa menyimpan suatu barang yang berlabel. Tipe data itu adalah label yang ada dibarang tersebut sedangkan variabel adalah kotak untuk nyimpen barang/data.
contoh deklarasi:"int kotak = 10".
int adalah tipe data/label yang menyatakan kotak tersebut berisi angka 10.

Ada beberapa tipe data dasar di Java, diantaranya:

  • String yang bisa nyimpen kata atau kalimat contoh: 'String nama = "budi";'. wajib nambahin tanda petik kalo buat data bertipe Stirng
  • int yang bisa nyimpen angka bulat contoh: 'int umur = 100;'
  • double yang bisa nyimpen angka desimal contoh: 'double tinggiBadan = 175.5;'
  • boolean Yang bisa nyimpen nilai true/false contoh: 'boolean sukaJva = true;'

Operators

Operator adalah simbol yang biasa digunain buat melakukan operasi matematika, ataupun logika pada nilai maupun variabel.

Ada beberapa operator di Java, diantaranya:

  1. Operator Aritmatika
    Operator ini buat ngitung kayak kalkulator:
    • + (pertambahan): '5 + 3 hasilnya 8'
    • - (pengurangan): '5 - 3 hasilnya 2'
    • * (perkalian): '5 * 3 hasilnya 15'
    • / (pembagian): '6 / 3 hasilnya 2'
    • % (modulus/sisa bagi): '5 % 3 hasilnya 2'
  2. Operator Penugasan
    Operator ini buat ngasih nyimpen ke variabel:
    • = (samadengan): 'int umur = 20'
    • += (tambah samadengan): 'int umur += 5 (umur jadi 25)'
    • -= (kurang samadengan): 'int umur -= 5 (umur jadi 15)'
    • *= (kali samadengan): 'int umur *= 2 (umur jadi 40)'
    • /= (bagi samadengan): 'int umur /= (umur jadi 10)'
  3. Operator Logika
    Operator ini buat logika (true/false):
    • && (dan): 'true && false hasilnya false'
    • || (atau): 'true || false hasilnya true'
    • ! (tidak): '!true hasilnya false'
  4. Operator Inkrement dan Dekrement
    Buat nambah atau ngurang nilai variabel secara simpel:
    • ++ (inkrement): 'int a = 5; a++;' (a jadi 6)
    • -- (decrement): 'int a = 5; a--;' (a jadi 4)

Berikut implementasinya pada kode java

                            
                                public class OperatorContoh {
                                    public static void main(String[] args) {
                                        int a = 5;
                                        int b = 3;
                                
                                        // Operator Aritmatika
                                        System.out.println("a + b = " + (a + b));
                                        System.out.println("a - b = " + (a - b));
                                        System.out.println("a * b = " + (a * b));
                                        System.out.println("a / b = " + (a / b));
                                        System.out.println("a % b = " + (a % b));
                                
                                        // Operator Penugasan
                                        int c = 10;
                                        c += 5;
                                        System.out.println("c += 5 -> c = " + c);
                                
                                        // Operator Perbandingan
                                        System.out.println("a == b: " + (a == b));
                                        System.out.println("a != b: " + (a != b));
                                        System.out.println("a > b: " + (a > b));
                                        System.out.println("a < b: " + (a < b));
                                
                                        // Operator Logika
                                        boolean x = true;
                                        boolean y = false;
                                        System.out.println("x && y: " + (x && y));
                                        System.out.println("x || y: " + (x || y));
                                        System.out.println("!x: " + (!x));
                                
                                        // Operator Inkrement dan Dekrement
                                        int d = 7;
                                        d++;
                                        System.out.println("d++ -> d = " + d);
                                        d--;
                                        System.out.println("d-- -> d = " + d);
                                    }
                                }

If...else statement

If-else statement adalah cara buat bikin keputusan di dalam programmu. Bayangin aja kamu lagi main game dan harus pilih jalan: kanan atau kiri. If-else statement ini ngebantu programmu buat milih jalan mana yang harus diambil berdasarkan kondisi tertentu.

Kurang lebih strukturnya kaya gini if(kondisi) {true } else { false }

Berikut adalah cara penggunaan if else statement pada program:

                        
                            int umur = 20;
                            if (umur >= 18) {
                                System.out.println("Kamu sudah dewasa!");
                            } else {
                                 System.out.println("Kamu masih di bawah umur.");
                            }

For-loop

Bayangin kamu disuruh ngetik maaf sama pacarmu 1000 kali,akan memakan waktu sangat lama jika mengetiknya satu per satu. Nah, untuk mengatasi hal ini kita bisa menggunakan for-loop. dengan for-loop kamu bisa mengulang kode yang sama secara praktis dan efisien sesuai dengan kebutuhan

Berikut adalah struktur kode for-loop

for (inisialisasi; kondisi; update) { // Kode yang diulang }
  • inisialisasi: inisialisasi adalah variabel yang digunakan untuk nge-set variabel awal
  • kondisi: Bagian ini dicek setiap kali loop mau dijalankan. Kalau true, kode di dalam loop dijalankan; kalau false, loop berhenti.
  • update: Bagian ini dijalankan setiap kali loop selesai satu putaran

Berikut ontoh penggunaan for-loop agar lebih jelas dan mengerti: 

                        
                            public class ForLoopContoh {
                                public static void main(String[] args) {
                                    for (int i = 0; i < 5; i++) {
                                        System.out.println("Perulangan ke-" + i);
                                    }
                                }
                            }

While-loop

While-loop mirip dengan for-loop, bedanya di penggunaan aja,perulangan for loop lebih terstruktur karena kamu perulangan harus ditentukan kapan mau berhenti, sedangkan untuk while loop cocok digunakan ketika kamu ga tau berapa kali kamu mau ngulang kayak lagi nonton video reels atau tiktok video akan terus berulang selagi kamu belum bosen atau sampe kamu scroll video selanjutnya

Berikut adalah struktur kode while-loop:

while (kondisi) { // Kode yang diulang }

Kondisi akan terus berulang selama kondisi bernilai true

Berikut adalah contoh penggunaan while-loop: 

                        
                            int i = 0;
                            while (i < 5) {
                                System.out.println("Perulangan ke-" + i);
                                i++;
                            }
                        
                    
                        
                            int i = 0;
                            while (i < 5) {
                                System.out.println("Perulangan ke-" + i);
                                if(i % 2 == 0) {
                                    break;
                                }
                            }

kamu pasti menyadari ada keyword break di kode diatas,nah fungsi keyword break itu buat ngeberhentiin perulangan tanpa perlu nyesuaiin dengan kondisi,bisa digunain juga buat perulangan for

Ada juga varian while yang lain yaitu do while-loop berikut cara penggunaannya: 

                        
                            int i = 0;
                            do {
                                System.out.println("Perulangan ke-" + i);
                                i++;
                                } while (i < 5);

Array

Jika variabel adalah kotak yang hanya bisa menyimpan satu data saja,maka array merupakan sebuah rak yang menyimpan banyak data sekaligus secara terstruktur

Untuk mendeklasrasi array kamu bisa seperti ini i[] a = new int[panjang data/kapasitas]

Contoh penggunaan: 

                        
                            // Deklarasi array tanpa inisialisasi(kapasitas data)
                            int[] angka;

                            // Deklarasi dan inisialisasi array
                            int[] nilai = new int[5];

                            // Deklarasi dan inisialisasi array dengan nilai awal
                            int[] skor = {100, 90, 85, 70, 60};

Untuk mengakses data array, indexnya dimulai dari angka 0 jadi semisal panjang data kamu 5 indeks terakhirnya itu 4 karena indeks dimulai dar 0 

                        
                            int[] skor = {100, 90, 85, 70, 60};
                            System.out.println("indeks ke 3: "+skor[3]);
                            int a = skor[0]
                            System.out.println("indeks ke 0: "+a);
